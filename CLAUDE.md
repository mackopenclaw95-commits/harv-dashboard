## Design skills available

Frontend work should respect Harv's established taste — see `docs/design/AWESOME_DESIGN.md` first (typography, color tokens, borders, motion, anti-slop checklist).

The following Claude Code design skills are installed under `.claude/skills/` and will auto-activate when relevant:

- **Impeccable** (18 commands): `impeccable`, `adapt`, `animate`, `audit`, `bolder`, `clarify`, `colorize`, `critique`, `delight`, `distill`, `harden`, `layout`, `optimize`, `overdrive`, `polish`, `quieter`, `shape`, `typeset` — anti-AI-slop design toolkit.
- **UI/UX Pro MAX**: `ui-ux-pro-max`, `design`, `design-system`, `ui-styling`, `brand`, `banner-design`, `slides` — industry-specific design intelligence.
- **Taste**: `taste-skill`, `gpt-tasteskill`, `minimalist-skill`, `brutalist-skill`, `soft-skill`, `redesign-skill`, `stitch-skill`, `output-skill` — anti-generic design philosophies.

When any skill's default guidance conflicts with `AWESOME_DESIGN.md`, Harv's doc wins for typography, color, borders, and motion.
